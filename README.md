# Student Record Management System Java

## Project Overview
Student Record Management System is a console-based Java application developed using Object-Oriented Programming and File Handling concepts.

This project helps in managing student academic records by allowing the user to add, view, search, and delete student details efficiently.

---

## Features

- Add New Student
- View All Students
- Search Student by ID
- Delete Student Record
- Data Stored in Text File
- Persistent File Handling Support

---

## Technologies Used

- Java
- OOP Concepts
- File Handling
- VS Code

---

## Files Included

- student.java
- recordmanager.java
- filehandler.java
- main.java
- students.txt
